# Risk Analysis Toolkit

A Python portfolio project for analysing sample trading and financial performance data.

This repository demonstrates practical risk and performance analytics relevant to finance, data analysis, trading operations and risk analyst roles. It uses sample trade data only and does not include any private trading logic, broker credentials, execution infrastructure or proprietary strategy parameters.

## Project Purpose

The aim of this project is to demonstrate how raw trade-level data can be converted into structured risk and performance outputs using Python.

The toolkit calculates:

- Win rate
- Expectancy
- Profit factor
- Equity curve behaviour
- Maximum drawdown
- Volatility
- Streak analysis
- Symbol-level performance
- Strategy-level performance

## Tools Used

- Python
- pandas
- NumPy
- matplotlib

## How To Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the script:

```bash
python risk_analysis.py
```

Generated files are saved into the `outputs/` folder.
